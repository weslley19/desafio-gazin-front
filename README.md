## Desafio React

Projeto para gerenciar desenvolvedores

- Criar
- Atualizar
- Visualizar
- Deletar

## Para utilizar o projeto:

```bash
# Clone o repositório
git clone https://github.com/weslley19/desafio-gazin-front.git

# Acesse o diretório
cd dir_do_projeto

# Instale as dependencias
npm install
yarn install

# Inicie o projeto
npm start
yarn start
```
